# 🧠 Text to All Text – VS Code Edition

**Text to All Text | VS Code Edition** is a powerful **all-in-one browser-based developer workspace** built using **HTML + TailwindCSS + JavaScript**.
It provides formatting tools, mini media player, spreadsheet editor, word processor, QR generator, web extractor, contact utilities, and more — inside a VS Code–style interface.

Developed by **Santhosh_A**

---

# 🚀 Features

## 🧾 Editor Workspace

* Multi-tab file editing
* File upload support
* Save edited files
* Download ZIP export
* HTML live preview mode
* VS Code–style UI experience

---

## 🧹 Format & Clean Tools

Supports formatting for:

* Plain Text
* Markdown
* JSON
* HTML
* CSS
* JavaScript
* Python
* C / C++

Extra utilities:

* Base64 encode / decode
* URL encode / decode
* Passage case conversion
* Hashtag generator
* YouTube title formatter

Cleaning operations:

* Trim edges
* Remove extra spaces
* Remove empty lines
* Remove duplicate lines
* Sort A → Z / Z → A
* Reverse line order

## 🎬 Mini Media Player (Queue System)

Features:

* Queue manager

## 📝 Built-in Word Processor
