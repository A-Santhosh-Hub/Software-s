# **📁 SanStructure Web v1.0 — Project Structure Generator**

**SanStructure Web** is a powerful, browser-based developer tool (now available as a standalone Windows Executable) designed to turn text-based folder hierarchies into actual project directories. Simply paste your tree-style structure, and SanStructure will generate a downloadable ZIP containing all your folders and files, complete with optional starter templates.

## **🚀 Desktop Version (.exe)**

We have successfully converted the SanStructure environment into a standalone **Windows Executable (.exe)** using a specialized Python wrapper.

* **Portable:** No installation required.  
* **Offline First:** Works entirely without an internet connection.  
* **High Performance:** Faster local processing of large project structures.

[**Download the Latest EXE Release here**](https://github.com/A-Santhosh-Hub/Software-s/blob/main/SanStructure/SanStructure.exe)

## **✨ Key Features**

* **Intelligent Tree Parsing:** Automatically recognizes common tree characters (├──, └──, │) and indentation depths.  
* **Smart Comment Stripping:** Paste structures directly from documentation; the parser automatically ignores inline comments (e.g., index.html \# Main entry).  
* **Starter Template Injection:** Automatically populates .html, .css, and .js files with modern boilerplate code.  
* **Activity History:** Locally saves your project templates. Never lose your work—even if you close the app or refresh the page.  
* **Modern Android 16 UI:** A clean, fluid user interface inspired by **Android 16** and **OriginOS 6**, featuring squircle containers and pill-shaped controls.  
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.  
* **Instant ZIP Export:** Generate and download your entire project structure in one click.

## **🛠 How to Use**

1. **Open the App:** Launch the .exe file on your Windows machine.  
2. **Input Structure:** Type or paste your folder tree into the **Editor**.  
   * *Example:*  
     /my-new-app  
     ├── index.html  
     ├── css/  
     │   └── style.css  
     └── js/  
         └── app.js

3. **Live Preview:** Watch the tree update in real-time in the **Preview** panel.  
4. **Save (Optional):** Click the **Save** button to store this template in your local history for future use.  
5. **Export:** Click **Export ZIP** to download your ready-to-code project structure.
